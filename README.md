# OptInvest 

## Description

Optimize your investment portfolio with OptInvest.



## Installation

1. clone the project from GitHub

2. install uv

```bash
pip install uv
```



3. create venv and activate it

```bash
uv venv
```

```bash
source .venv/bin/activate
```


4. install dependencies in the venv

```bash
uv add -r requirements.txt
```
